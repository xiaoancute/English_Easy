# v0.1.0-rc8

这一版补上学习状态的第一层闭环，让词库不再只是一个静态清单。

## Changes

- 新增持久化学习状态表，区分学习中、已掌握、已跳过。
- 词库页的今日学习列表增加开始学习、跳过操作。
- 已跳过的词会进入独立区块，可恢复到未学习状态。
- 查词成功后会把新词纳入学习中，但不会覆盖已跳过或已掌握状态。
- 复习满强度后标记为已掌握，已掌握词不再进入今日学习队列。
- 旧缓存卡片在数据库升级时回填为学习中，避免升级后进度消失。
- 修正短语词库进度统计的空格归一化问题。

## Verification

- GitHub Actions `Build` workflow passed on `rc-layout-build-check`.
- Unit tests, debug APK assembly, and release APK assembly all passed remotely.
