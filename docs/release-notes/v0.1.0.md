# English Easy v0.1.0

第一个自用正式版。

English Easy 是面向中文学习者的英语概念还原器：把英文词和短语背后的核心画面、典型场景和常见误解讲清楚，帮助用户从“中文释义列表”转向“可迁移的概念理解”。

## Included

- OpenAI-compatible BYOK 配置：Base URL、模型和 API Key 保存在本机。
- 单词、固定短语、普通自由组合的概念卡生成。
- 核心概念、中文逼近、典型场景、错误直觉、多义分支展示。
- 本地历史、收藏、笔记、复制、分享和重新生成。
- “我的输出”区域：查完后写一句自己的英文例句。
- Study 学习页：今日新词、到期复习、薄弱词和词库选择。
- 小学、初中、高中、高考词库包，以及本地学习状态。
- 跳过/恢复、学习中/已掌握、轻量复习调度。
- Release APK 由 GitHub Actions 构建并上传。

## Product boundary

这一版是“概念理解加速器”，不是完整英语学习系统。

它适合：

- 查容易误解的词和短语。
- 看核心画面和典型语境。
- 写自己的例句，形成轻量输出闭环。
- 用本地词库安排少量新词和复习。

它不承诺：

- 替代真实阅读、听力、口语和写作训练。
- 提供完整课程、考试训练或外教式陪练。
- 内置公共 API Key。
- 保证所有模型输出质量一致。

## Changes since rc9

- 首页文案从“查询工具”调整为“核心画面 -> 自己例句”的学习动作。
- 概念卡把“我的例句”收敛为“我的输出”，强调主动造句。
- 学习页新词入口改为“查概念并造句”。
- 词库页和今日概览文案统一到“新词 + 复习 + 输出”的闭环。
- 保持 UI 微调范围，不新增页面、不改数据库、不引入第三方指南内容。
