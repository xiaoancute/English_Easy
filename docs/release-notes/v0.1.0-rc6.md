# English Easy v0.1.0-rc6

Sixth signed release candidate for English Easy.

## Changes since rc5

- Added 1,987 junior entries from the English headwords in `KyleBing/english-vocabulary`.
- Added 3,743 senior entries from the English headwords in `KyleBing/english-vocabulary`.
- Updated vocabulary source documentation to reflect README-based reuse.
- Updated third-party notices for `KyleBing/english-vocabulary`.
- Extended vocabulary asset tests to cover junior and senior source entries.

## Notes

- This is still an RC build.
- Junior and senior packs use only English headwords from a README-permitted learning resource.
- Definitions, translations, phrases, and examples from `KyleBing/english-vocabulary` are not bundled.
- `KyleBing/english-vocabulary` itself is derived from `kajweb/dict`, whose README says it was crawled from a word-learning app; this is why English Easy keeps the import narrow.
