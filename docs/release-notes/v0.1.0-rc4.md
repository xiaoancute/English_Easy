# English Easy v0.1.0-rc4

Fourth signed release candidate for English Easy.

## Changes since rc3

- Added a student vocabulary packs MVP in the Study tab.
- Added four vocabulary stages: primary, junior, senior, and gaokao.
- Added a small seed vocabulary asset for validating the learning flow.
- Added learned-count progress for each vocabulary pack.
- Added pack filtering so learned words are hidden from the active pack list.
- Added word tap-through from vocabulary packs back to concept lookup.
- Added unit tests for vocabulary catalog parsing and grouping.

## Notes

- This is still an RC build.
- The vocabulary list is a seed sample, not a complete official curriculum list.
- Concept cards are still generated by the selected AI provider; the vocabulary packs only define what to learn next.
