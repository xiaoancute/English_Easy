# English Easy v0.1.6

这一版集中修正界面层级、响应式布局和容易误操作的交互。

## Changes

- 统一浅色与深色配色，减少互相冲突的强调色。
- 首页支持横屏双栏，系统字体放大后搜索框不再裁切。
- 压缩查询结果顶部区域，把分享、复制和重新生成收进更多菜单。
- 精简首页、学习页和概念卡中的重复说明文案。
- 设置页把 API Key、测试和保存放到首屏，Base URL 与模型移入高级设置。
- 服务商选择增加明确的选中标记和无障碍选择语义。
- 历史列表改为紧凑布局，单项删除支持撤销，清空历史需要确认。
- 空状态增加直接行动入口，并修复收藏页图标和查词跳转。
- 修复查词导航、缓存身份和学习队列相关问题。

## Verification

- GitHub Actions 完成单元测试、debug APK 和 release APK 构建。
- 使用 Actions 产出的 APK 验证竖屏、横屏、130% 字体、浅色、深色、查询结果、设置和历史删除撤销。
