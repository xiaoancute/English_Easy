# English Easy v0.1.0-rc5

Fifth signed release candidate for English Easy.

## Changes since rc4

- Expanded the bundled student vocabulary data.
- Added 831 primary-school entries from an MIT-licensed source.
- Added 3,668 gaokao entries from an MIT-licensed gaokao 3500 source.
- Kept junior and senior packs as small seed placeholders until a clearly licensed source is found.
- Added source tracing for bundled vocabulary entries.
- Added third-party license notices for bundled vocabulary sources.
- Added vocabulary asset tests for source coverage and phrase entries.

## Notes

- This is still an RC build.
- Primary and gaokao packs now use explicit open-source sources.
- Junior and senior packs are not complete yet.
- `KyleBing/english-vocabulary` was not bundled because no repository license is declared.
