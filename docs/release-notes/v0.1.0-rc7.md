# English Easy v0.1.0-rc7

Seventh signed release candidate for English Easy.

## Changes since rc6

- Added a minimal "today learning" flow in the Study vocabulary tab.
- Selecting a vocabulary pack now surfaces the first 10 unlearned words as today's learning list.
- Today words are normalized and deduplicated before display.
- Remaining unlearned words are still available below today's list.
- Added unit coverage for the today learning planner.

## Notes

- This is still an RC build.
- Today's list is intentionally simple and deterministic for now.
- A richer learning-state model can be added later without changing the current vocabulary data.
