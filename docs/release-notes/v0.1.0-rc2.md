# English Easy v0.1.0-rc2

Second signed release candidate for English Easy.

## Changes since rc1

- Added explicit app version, version code, prompt version, and license information to the About screen.
- Simplified empty states for History and Favorites.
- Release workflow now uses checked-in release notes when available.

## Included

- Concept cards for English words and phrases.
- Support for words, fixed phrases, and free combinations.
- BYOK configuration for OpenAI-compatible endpoints.
- Local history, favorites, notes, copy, and regenerate.
- Local encrypted API key storage.
- Signed APK release through GitHub Actions.

## Notes

- This is still an RC build.
- The app does not include a shared API key. You need your own provider key.
- The app sends lookup requests directly to the Base URL configured on your device.
