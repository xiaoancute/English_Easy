# v0.1.0-rc9

这一版集中处理正式版前的 UI 质感和学习闭环可见性。

## Changes

- 统一查词、概念卡、学习、词库、设置页的视觉层级。
- 降低卡片堆叠感，让概念卡更像结构化学习笔记。
- 强化学习页当前任务优先级，减少数据看板感。
- 新增学习任务查词成功/失败、跳过、恢复、设置保存等反馈。
- 修正发布说明中“所有查词成功都会计入学习”的过时表述。

## Verification

- GitHub Actions `Build` workflow passed on `rc-layout-build-check`.
- Unit tests, debug APK assembly, and release APK assembly passed remotely.
- Screenshot review covered Home idle, Home success, Study today task, Vocabulary pack list, and Settings provider setup.
