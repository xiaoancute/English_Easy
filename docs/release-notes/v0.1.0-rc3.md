# English Easy v0.1.0-rc3

Third signed release candidate for English Easy.

## Changes since rc2

- Added a lightweight review queue.
- Added a Study tab.
- Added three review outcomes: forgot, vague, remembered.
- Added review scheduling state to cached concept cards.
- Added Room migration from database version 3 to 4.
- Added unit tests for review scheduling rules.

## Review schedule

- Forgot: due again in 10 minutes.
- Vague: due again in 1 day.
- Remembered: increasing intervals of 1, 3, 7, 14, then 30 days.

## Notes

- This is still an RC build.
- Review data is local-only.
- Existing cached cards are added to the review queue during database migration.
