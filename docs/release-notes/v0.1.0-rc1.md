# English Easy v0.1.0-rc1

First signed release candidate for English Easy.

## Included

- Concept cards for English words and phrases.
- Support for words, fixed phrases, and free combinations.
- BYOK configuration for OpenAI-compatible endpoints.
- Local history, favorites, notes, copy, and regenerate.
- Local encrypted API key storage.
- GitHub Actions release build with signed APK output.

## Notes

- This is an RC build, not a final stable release.
- The app does not include a shared API key. You need your own provider key.
- The app sends lookup requests directly to the Base URL configured on your device.
- Current AI output quality depends on the selected model and prompt stability.

## Manual RC checklist

- First launch.
- Save Base URL, model, and API Key.
- Query `spring`.
- Query `break the ice`.
- Query `red apple`.
- Favorite a card.
- Add a note.
- Copy a card.
- Regenerate a card.
- Restart the app and verify history, favorites, and notes persist.
